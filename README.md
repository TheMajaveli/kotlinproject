# Projet Kotlin

## Objectif
L’objectif de ce projet est de mettre en pratique l’ensemble des compétences acquises durant le cours : Kotlin pour Android, Jetpack Compose, navigation, stockage local avec Room et bonnes pratiques de sécurité.

Vous devez concevoir et développer une application mobile Android fonctionnelle.

---

## Organisation
Chaque projet doit être réalisé par **1 à 3 personnes maximum**.

Les contributions doivent être documentées via **GitHub/GitLab** avec des **commits réguliers** pour chaque membre.

---

## Exigences minimales

Votre application doit inclure :

### 1. Interface utilisateur moderne avec Jetpack Compose
- Chaque membre de l’équipe doit créer **au moins deux écrans différents**
- Composants interactifs :
    - Boutons
    - Formulaires
    - Listes
    - etc.

---

### 2. Navigation entre écrans
- Implémenter la navigation avec le **Navigation Component**

---

### 3. Stockage local avec Room
- Création d’une **base de données locale**
- Sauvegarde de **données persistantes**

---

### 4. Sécurité mobile et bonnes pratiques
- Gestion des permissions (ex : caméra, stockage)
- Stockage sécurisé des informations sensibles si nécessaire

---

## Livrable
- Code source complet sur **GitHub**
- Commits réguliers pour chaque membre
